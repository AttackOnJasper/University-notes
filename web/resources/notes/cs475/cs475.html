Lecture 2 (2018-05-07)

Objectives

  • \displaystyle \bar P \vec x = \vec x, \bar P = \alpha P + (1 - \alpha)\vec v 1^T
  • \displaystyle (I - \alpha P)\vec x = (1 - \alpha)\vec v

Matlab

  • j-th column of A - \displaystyle A_{:,j}
  • i-th row: \displaystyle A_{i,:}

Matrix Multiplication

  • \displaystyle A \in \Re^{m*n}, B \in \Re^{n*p}, C = A * B \in \Re^{m*p} complexity O(mnp)
  • Strassen's alg. \displaystyle O(n^{2.89})

Matrix Inverse

  • A is invertable if and only if \displaystyle A \in \Re^{n*n} and \displaystyle A^{-1}*A = A * A^{-1} = I
  • Thm (Sherman-Morrisen-Woodbury): \displaystyle A\in\Re^{n*n}, B\in\Re^{m*m}, u\in\Re^{n*m}, V\in\Re^{n*m}, (A + uBV)^{-1} = A^{-1} - A^{-1}u(B^{-1}+v^TA^{-1}u)^{-1}vA^{-1}
    • In particular, \displaystyle (A + \vec u \vec v)^{-1} = A^{-1} - \frac{1}{1+v^TA^{-1}u}A^{-1}uv^TA^{-1}
    • Proof: verify that the product is 1

Goal: solve \displaystyle A\vec x = \vec b, A \in \Re^{n*n} , A is invertible

  • $$ x = A^{-1}b
  • e.g. \displaystyle \begin{cases} 2x + y + 2z = 1 \\ 4x + 2y -z = 2 \\ x + y + 2z = 3 \end{cases}
  • to solve a linear system, try to transform the matrix \displaystyle \begin{bmatrix} 2 & 1 & 2 \\ 4 & 2 & -1 \\ 1 & 1 & 2 \end{bmatrix} to an upper triangular matrix
  • Psedo code for solving lower triangular matrix O(n^2)
    • Problem: division of \displaystyle l_{i,i} could be 0. Solution: add a small number to all diagonals to avoid such issues / pivoting
  • During implementation, try to use vector instead of for-loop for calculation (avoid loop as much as possible)
  • Gaussian Elimination psedo code
    • Input: \displaystyle A \in \Re^{n*n}, \vec b \in \Re^{n*1}
    • output: in-place transform A to upper triangular matrix

    • code:

      for j = 1,2 ... n-1 do
    for i = j + 1, ..., n do
    $$  a_{i,j} <- a_{i,j} / a_{j,j} $$
    $$  \vec a_{i,j+1:n} <- \vec a_{i,j+1:n} - a_{ij} $$
    • still have the problem of division
      • how to guarantee that the diagnals are never zero?
      • Answer: compute the determinent: \displaystyle det(A) = \sum_{\sigma}(-1)^{sgn(\sigma)} * \prod^{n}_{i=1} A_{i, \sigma(i)}

Thm (LU factorization): let and u be the strict lower triangular and upper triangular of the output of GE, then A = L*u, L = + I

Solve linear system

  • note: don't use inverse ever!
  • \displaystyle \begin{aligned} A\vec x = \vec b, A &= L*u \\ L*u\vec x &= \vec b \\ L * \vec y &= \vec b (forward_solve) \\ u \vec x &= \vec y (backward_solve) \end{aligned} time complexity \displaystyle O(n^3)
