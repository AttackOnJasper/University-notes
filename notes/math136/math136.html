Math 136 Review Linear Algebra

Lecture 1

  • Vector n-tuple of real numbers
    • Elements in a vector are called entries, components, or coordinates
    • Column vector / row vector
  • Collinear one vector is a scalar multiple of the other
  • Linear combinations of vectors a sum of scalar multiples of vectors
  • Vectors within 3-d can be represented as Direct Line Segment (starting from 0 and end from (\displaystyle x_1, x_2, x_3 ))

Vector Equation of the line L: \displaystyle \vec x = \vec a + t\vec d , where d is direction vector

2 lines \displaystyle L_{1} and \displaystyle L_{2} are parallel if direction vector \displaystyle d_{1} is a scalar multiple of direction vector \displaystyle d_{2}

Lecture 2

  • Linearly independent set (each vector is not a linear combination of any other vectors in the set)
    • each L.I. set cannot contain 0
    • If a vector \displaystyle \vec v in U is a linear combination of others, we say \displaystyle \vec v is redundant in U
  • \displaystyle z = s\vec v + t\vec w + \vec b represents a plane in 3-d
  • If \displaystyle \vec b \in \Re^n , then \displaystyle \vec z = \vec b + a_1\vec v_1 + a_2\vec v_2 + ... + a_k\vec v_k where k < n is k-flat in \displaystyle \Re^n.
    • Plane in \displaystyle \Re^3 is 2-flat
    • Hyperplane (\displaystyle \Re^n where n > 3): (n-1)-flat

Lecture 3

  • \displaystyle \Re^n is not only the set of all n-tuples of real numbers but also as a vector space
  • a subset W of \displaystyle \Re^n is a subspace of \displaystyle \Re^n if W is a non-empty subset (i.e. has some vectors and contains 0) and is 'closed under scalar multiplication' and 'closed under addition' (i.e. if \displaystyle \vec u, \vec w \in W , then \displaystyle \alpha \vec u \in W and \displaystyle \vec u + \vec w \in W )
    • Straight lines \displaystyle \vec x = t\vec v and planes \displaystyle \vec x = s\vec v + t\vec w are subspaces in \displaystyle \Re^3

Span

  • \displaystyle S = span\{v_{1}, v_{2}, ..., v_{k}\} means S is the set of all linear combinations of the vectors \displaystyle \vec v_{1} ... \vec v_{k}
    • the vectors \displaystyle \vec v_{1} ... \vec v_{k} is a spanning family of S or a spanning set of S
    • The set {\displaystyle e_{1}, e_{2}, ... e_{n} } spans \displaystyle \Re^n
    • Span is a subspace, and every subspace S of \displaystyle \Re^n has a finite spanning family
  • If S = span{\displaystyle v_{1}, v_{2}, ..., v_{k} } where {\displaystyle v_{1}, v_{2}, ..., v_{k} } is L.I., then {\displaystyle v_{1}, v_{2}, ..., v_{k} } is a basis of subspace S
    • {\displaystyle e_{1}, e_{2}, ... e_{n} } is the standard basis of \displaystyle \Re^n

Lecture 4

Dot product

Norm (length) of x, denoted by \displaystyle \lVert x \rVert , is \displaystyle \lVert x \rVert = \sqrt{x*x}

  • for every \displaystyle \vec x , \displaystyle \lVert x \rVert \geqslant 0
  • \displaystyle \lVert x + y \rVert \leqslant \lVert x \rVert + \lVert y \rVert
  • Cauchy-Schwartz inequality: \displaystyle \lVert x * y \rVert \leqslant \lVert x \rVert * \lVert y \rVert
  • Unit vector in direction of a is \displaystyle \frac{a}{\lVert a \rVert}
    • if b has the same direction as a, then \displaystyle \frac{a}{\lVert a \rVert} = \frac{b}{\lVert b \rVert}
  • Let a and b be 2 vectors in \displaystyle \Re^n , distance between 2 vectors is \displaystyle \lVert b - a \rVert
    • angle between a and b: \displaystyle \cos\theta = \frac{a * b}{\lVert b \rVert * \lVert a \rVert}

Orthogonal / orthogonal set

  • orthonormal set: orthogonal set & \displaystyle \lVert v_{i} \rVert = 1 for i = 1 to k
  • In \displaystyle \Re^{2}, \Re^{3} , orthogonal non-zero vectors are perpendicular

Lecture 5 Cross Product in \displaystyle \Re^{3}

  • Properties
    • \displaystyle a \times b is orthogonal to both a and b
    • \displaystyle a \times 0 = 0 = 0 \times a
    • \displaystyle a \times b = - (b \times a)
    • \displaystyle \lVert a \times b \rVert = \lVert b \rVert * \lVert a \rVert * \sin\theta
  • Right-hand rule

Lecture 6 Planes in \displaystyle \Re^{3}

  • P(n,a) as being all vectors s.t. \displaystyle x - a \perp n
    • n is the normal to the plane
  • if \displaystyle \Re^{n} where n > 3, then the set of vectors is called a hyperplane
  • Scalar equation of the plane: \displaystyle n_1x_1 + n_2x_2 + n_3x_3 = d

Lecture 8 Systems of linear equations

  • Linear equations \displaystyle a_1x_1 + a_2x_1 + ... + a_nx_n = b , where x's are variables, a's are coefficients, b is constant
  • Systems of linear equations
  • 2 systems of linear equations are equivalent systems if they have the same set of solutions
  • Coefficient matrix / augmented coefficient matrix
  • Pivot: the first non-zero number in a row of a matrix
  • Reduced row echelon form (RREF)
    • All pivots are leading-1s
    • A leading-1 is the only entry in its column which is not zero

Lecture 9 Solutions of RREF systems

  • Thm: if a system of linear equatins has more than 1 solution, then it has infinitely many solutions
  • Basic unknowns / free unknowns
  • Rank of coefficient matrix A = number of basic variables

Lecture 12 Matrices and matrix algebra

  • \displaystyle A \in \Re^{m*n} means m rows & n columns
  • row-vectors, column-vectors, column matrix, row matrix
  • Transpose
  • Symmetric: \displaystyle A = A^T
  • Trace of A is the sum of the numbers on the main diagonal
  • Matrix-vector multiplication
    • allows us express a system of m linear equations with n unknowns
    • can be seen as a linear combination of column vectors
